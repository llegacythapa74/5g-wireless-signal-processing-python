# Digital Signal Processing

Bitwise and binary operations used in wireless communications.

## What this covers
- AND, OR, XOR operations on binary sequences
- Hamming distance — measuring bit errors between two sequences

## Why it matters
Hamming distance is how we measure channel errors in practice.
The more bits that flip, the worse the channel quality.
